# SAVELEV-AI
Показываю, внедрять нейросети и AI автоматизацию,создавать и монетизировать ИИ-контента.

Здесь эксперименты, промпты, скрипты, заметки и всё, что связано с ИИ.

## Что внутри
- Промпты и шаблоны для Grok / ChatGPT / Claude
- Скрипты для автоматизации (Python, JS и др.)
- Заметки по обходам ограничений, тюнингу моделей
- Идеи и roadmap
